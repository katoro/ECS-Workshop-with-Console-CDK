# IAM 생성

## 개요

ECS 실습을 위한 IAM 역할 및 정책을 생성합니다.

## 실습

* Cloud9 콘솔에서 아래 명령어를 수행합니다.

```
cd ~/environment/ecs-ec2-cdk/ecs-iam-role/
cdk deploy
```
